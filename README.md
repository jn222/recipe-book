# recipe-book

## Requirements

- Searchable recipes on homepage, displayed in list
- Recipe display page

## Assumptions

- Recipes should have a title, ingredients list, picture url, info blurb, and steps
  - Typing definition:
    - title: string
    - info: string
    - ingredients: string[]
    - picture: string
    - steps: string[]
